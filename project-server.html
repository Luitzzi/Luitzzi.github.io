Entstehungsgeschichte

History of origins

Geschafft!!... Die vorerst finale Version der Webseite war fertig.
Jedoch stand ich damals direkt vor dem nächsten Problem: Wie bekomme ich diese nun ins Internet?
Zuerst hostete ich die Webseite lokal in einer virtuellen Maschiene auf meinem PC. Leider konnte dies aus folgenden Gründen nicht die Lösung sein. Einerseits wollte ich nicht mein Heimnetzwerk "öffnen" und andererseits wären die Stromkosten viel zu hoch...
Deshalb wählte ich einen gemieteten virtuellen Server von Hetzner und richtete diesen folgendermaßen ein:

Finally... I made it!! I finished my final version of the website. However, the next problem emerged immediately afterwards: How do I make it accessible to the internet?
First of all I hosted the website locally on a virtual machine on my computer. Considering the electricity costs and the necessity of opening my home-network to the internet, this could only be a temporary solution. Due to these facts, I switched to a rented virtual server by Hetzner. Following is a short description of my server setup and a more detailed short video.
